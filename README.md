# mooreasgd

A package to clean, organize, and analyze data from loggers placed near submarine groundwater discharge on coral reefs in Mo'orea, French Polynesia for research in the Silbiger Lab, CSUN.

### Available functions
- CT_roundup  
- CT_cleanup  
- CT_one_cal  
- CT_two_cal  
- WL_cleanup  
- pH_cleanup
